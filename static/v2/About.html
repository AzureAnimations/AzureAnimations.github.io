About this project

What powers Azure Animations?

A static, dependency-light web app that turns Azure concepts — storage redundancy, Foundry agents, vision-enabled GenAI — into interactive, animated stories you can step through in the browser. No build step. No backend. Just HTML, vanilla JavaScript, and a handful of well-known open-source libraries.

The stack

Built with familiar tools

Every page in this site is built from the same compact set of technologies. Hover a card to see it react.

HTML5 & CSS3

Semantic markup, CSS custom properties, responsive grids — no framework runtime.

Vanilla JavaScript

Plain ES2017+ scripts in lib/js. No bundler, no transpiler, no React.

Microsoft Atlas

The same design system that powers Microsoft Learn — typography, tokens, components.

JavaScript Animations

Custom JavaScript drives every timeline, tween, and step transition.

Fluent UI Emoji

Microsoft's open-source emoji set, used as illustrations across Foundry pages.

RecordRTC

Optional in-browser screen / video recording on the Foundry & Vision pages.

JSZip

Bundles recorded clips and assets into downloadable .zip files client-side.

100% open source

Every dependency below ships under a permissive license. Verified, linkable, auditable.

Is it safe to use?

License summary at a glance

Mostly MIT. Atlas, RecordRTC, JSZip, Fluent UI Emoji, and the project's own code are all MIT-licensed — meaning you can use, modify, and redistribute them in commercial or personal projects, as long as you keep the copyright notice.
Heads up — GSAP. GSAP uses its own Standard "No Charge" License, not MIT. It is free for the vast majority of uses (including this educational site). A paid Business Green license is required only if you sell a product whose primary value is the animation itself (e.g. a paid game or a paid app sold to end-users). For internal tools, demos, and most commercial websites, the free license is sufficient.
Static & private by design. No analytics, no cookies, no backend. Everything runs in your browser, and CDN assets are pinned to specific versions for reproducibility.
